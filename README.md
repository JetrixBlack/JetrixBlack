## 🏆 Proyectos Destacados

### 🌐 Portafolio Web

Portafolio profesional diseñado para mostrar mis proyectos y habilidades. Una plataforma moderna y responsiva que destaca mi experiencia en desarrollo web frontend y animaciones interactivas.

**Tecnologías utilizadas:**

- Framer Motion y GSAP para animaciones

**Destacados:** 
- ✅ Diseño moderno
- ✅ Optimización de rendimiento
- ✅ SEO amigable
- ✅ Componentes interactivos

---

### 💰 Presupuesto Personal

Aplicación de gestión de presupuestos personales que ayuda a los usuarios a controlar sus gastos, ingresos y ahorros de forma intuitiva, desarrollada bajo una arquitectura limpia y segura.

**Tecnologías utilizadas:** 
- PHP Nativo (Arquitectura MVC), MySQL/MariaDB, Tailwind CSS, JavaScript

**Destacados:**
- ✅ Dashboard interactivo
- ✅ Reportes de gastos
- ✅ Autenticación segura (encriptación bcrypt, mitigación CSRF, protección contra fuerza bruta)
- ✅ Base de datos relacional

---

### 🏥 Sistema de Citas Médicas

Plataforma de gestión clínica para la administración de triaje, consultas médicas y recepción. Diseñada para mejorar la eficiencia y el control de accesos en consultorios médicos.

**Tecnologías utilizadas:** 
- PHP Nativo (Arquitectura MVC), PostgreSQL, HTML/CSS, JavaScript

**Destacados:**
- ✅ Gestión de citas
- ✅ Historial de pacientes
- ✅ Control de roles (Administrador, Recepción, Enfermería, Médico)
- ✅ Alta seguridad (protección contra Session Fixation y Clickjacking)
